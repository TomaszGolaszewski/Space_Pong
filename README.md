# Space_Pong
Play 'Space pong' with a friend! Who will win the Empire or the Rebel Alliance?

## Instruction
- To play 'Space pong', run the file _space_pong.py_
- Controls: 
  - Player I: WASD+LCTRL
  - Player II: Arrows+Space
